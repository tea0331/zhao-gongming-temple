## 三、充值渠道方案（双轨制）

### 3.1 国内轨道 → 上海公司 + 郑州个独

```
用户（微信/小红书登录）
    ↓
微信支付 / 支付宝
    ↓
上海公司（一般纳税人）收款
    ↓ 对公
服务成本、服务器等（6%增值税可抵扣）
    ↓ 利润转移
郑州个人独资（核定征收）
    ↓ 综合税负约3-5%
老板个人账户
```

**关键操作**：
- 上海公司作为**签约主体**，用户协议、发票都走上海
- 上海公司向郑州个独支付"技术服务费/咨询费"（有真实业务支撑）
- 郑州个独核定征收，综合税负约3-5%（远低于上海25%企业所得税+20%分红税）
- ⚠️ 利润转移比例需合理，一般不超过总收入的70%，要有合同和交付物支撑

**国内支付对接**：

| 渠道 | 费率 | 对接方式 | 适合阶段 |
|------|------|---------|---------|
| 微信支付 | 0.6% | 微信商户平台 | MVP阶段 |
| 支付宝 | 0.6% | 支付宝开放平台 | MVP阶段 |
| 聚合支付（收钱吧/拉卡拉） | 0.38-0.6% | API对接 | 量大了切 |

### 3.2 国际轨道 → Stripe + 加密货币

```
海外用户（Facebook/Twitter/Google登录）
    ↓
Stripe（信用卡/Apple Pay/Google Pay）
    ↓
Stripe账户 → 上海公司外币对公账户
    ↓ 或
加密货币（USDT/USDC）
    ↓
郑州个独/个人钱包
```

**国际支付对接**：

| 渠道 | 费率 | 特点 | 适合阶段 |
|------|------|------|---------|
| **Stripe** | 2.9%+$0.30 | 全球信用卡、最成熟 | 首选，MVP即可接 |
| **Lemon Squeezy** | 5%+$0.50 | 不需要公司主体，替你收款 | 没有Stripe时用 |
| **Crevio** | 5% | 专为数字产品，支持支付宝 | 补充渠道 |
| **加密货币（USDT/USDC）** | 0-1% | 无国界、隐私友好 | 技术向用户 |

**Stripe开户路径**：
- 方案A：上海公司直接申请Stripe China（需外管局备案）
- 方案B：用香港主体开Stripe（香港公司注册约¥3000）
- 方案C：先走Lemon Squeezy（无需主体，当天上线），量大了再切Stripe

## 五、免费建站方案

| 组件 | 方案 | 费用 | 说明 |
|------|------|------|------|
| **前端托管** | Cloudflare Pages | $0 | 全球CDN，无限带宽，国内速度OK |
| **域名** | Freenom（.tk/.ml）或 buy便宜.xyz | $0-1/年 | .xyz域名首年约$1 |

| **数据库** | Cloudflare D1（SQLite） | $0（5GB免费） | 用户数据、签诗、订单 |
| **用户认证** | Auth.js（自建） | $0 | 开源自托管 |
| **图片存储** | Cloudflare R2 | $0（10GB免费） | 神像图、动画资源 |
| **支付** | Stripe/Lemon Squeezy | 按交易收费 | 无月费 |
| **邮件** | Resend | $0（100封/天免费） | 注册确认、签诗推送 |

**总成本：≈ $0/月**（纯免费方案跑MVP）

## 七、公司架构与资金流

```
┌─────────────────────────────────────────────────┐
│                    赵公明财神庙                      │
│                 （网站/App产品）                    │
└───────────┬─────────────────────┬───────────────┘
            │                     │
    ┌───────▼───────┐    ┌───────▼───────┐
    │  上海公司       │    │  郑州个独       │
    │  （一般纳税人）   │    │  （核定征收）    │
    │               │    │               │
    │  • 国内收款     │    │  • 技术服务     │
    │  • 开票        │    │  • 利润归集     │
    │  • 合规运营     │    │  • 低税提现     │
    │  • Stripe外币  │    │  • 加密货币     │
    └───────┬───────┘    └───────┬───────┘
            │                     │
            │   技术服务费（合理比例）  │
            └──────────►──────────┘
                        │
                  ┌─────▼─────┐
                  │  老板个人    │
                  │  综合税负3-5%│
                  └───────────┘
```

**关键数字**：
- 上海公司企业所得税：25%（小型微利企业可享5%优惠税率至2027年底）
- 上海公司利润→分红：20%个人所得税
- 郑州个独核定征收：综合税负约3-5%
- **节税效果**：走郑州个独比直接分红节省约15-20个百分点
